# Enterprise Knowledge AI System (Advanced RAG)

An intelligent AI-powered system designed to analyze, compare, and summarize multiple documents with high accuracy and zero hallucination.

This system acts as an **AI Data Analyst**, enabling users to extract insights from business, HR, financial, and technical documents in a structured and explainable way.

---

## 🚀 Live Demo

👉 **Try the application here:**
🔗 https://enterprise-knowledge-ai-pvwzttt57sdkxcbj7vcx3u.streamlit.app/

---

## 🎯 Key Features

* 📂 Multi-format document support (PDF, TXT, CSV, Excel)
* 🤖 Context-aware Question Answering
* 📊 Intelligent Dataset Comparison
* 🧠 Structured Multi-document Summarization
* 📚 Explainable AI with source citations
* ❌ No hallucination (strict context-based answers)
* ⚡ Fast retrieval using vector search (FAISS)

---

## 🧠 How It Works

1. Upload multiple documents
2. System processes and converts them into embeddings
3. Retrieves relevant context using vector search
4. Generates accurate answers using LLM
5. Provides sources for transparency

